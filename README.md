# Time complexity: Sum all integers of an interval

## System requeriments (recommended)
* Ubuntu 20.04 LTS

## 1. Configure and run the app

1.1 Install the latest version of Python 3 and verify their version.
```
$ sudo apt install python3
```
```
$ python3 --version
```

1.2 Run the app
```
$ python3 app.py
```

## 2. Output
e.g.
```

------------------------------------------------------
```
